Quiénes Somos

Este proyecto académico fue desarrollado con el propósito de aplicar los conocimientos adquiridos en la asignatura de Desarrollo Web. La página representa una propuesta moderna enfocada en el diseño y construcción de sitios web utilizando HTML5, CSS3 y Bootstrap. Se aplican buenas prácticas de organización mediante etiquetas semánticas, diseño adaptable y elementos multimedia que mejoran la experiencia del usuario. El objetivo es demostrar habilidades en el desarrollo de páginas web profesionales, capaces de visualizarse correctamente en computadoras, tabletas y dispositivos móviles. Además, se busca fortalecer el aprendizaje práctico mediante la implementación de herramientas actuales utilizadas en el desarrollo web moderno.
