Inicio

Bienvenido a nuestro proyecto de Desarrollo Web. Este sitio tiene como objetivo presentar información sobre la creación de páginas web modernas utilizando HTML5, CSS y buenas prácticas de diseño para ofrecer experiencias atractivas a los usuarios.

Quiénes Somos

Este proyecto académico fue desarrollado con el propósito de aplicar los conocimientos adquiridos en la asignatura de Desarrollo Web. La página representa una propuesta de sistema web orientada a la organización y presentación de información digital de manera clara y estructurada. A través de este trabajo se emplean etiquetas semánticas de HTML5 para mejorar la accesibilidad, la organización del contenido y la experiencia del usuario. Además, se incorporan elementos multimedia como imágenes y videos para enriquecer la presentación del proyecto. El objetivo principal es demostrar la capacidad de diseñar páginas web funcionales, atractivas y adaptables a diferentes dispositivos, siguiendo estándares actuales de desarrollo web y fomentando el aprendizaje práctico de las tecnologías utilizadas en la construcción de sitios web modernos.
